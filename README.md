# Guessing-Game-using-sockets-in-python
This is a guessing game that uses sockets in python

# The Game Consists of 3 Difficulties

a. easy (1-50)

b. medium (1-100)

c. hard (1-500)

The numbers in the parentheses is the range of the numbers to guess from
